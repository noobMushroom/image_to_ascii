# Ascii Image generator 


This project is in the early phase.  You can add you image absolute path in the file ```src/bin/main.rs``` by uncommenting the ```let img_path``` line and adding your image path there.

#### About
So far this program can detect your terminal size and then print the image according to the terminal size. Each character represent 1 pixel so size your terminal according to that.
